🔐 Hashing in Data Structures & Algorithms

This repository contains my practice and solutions for Hashing-based DSA problems, mainly from platforms like LeetCode.
The goal is to build strong problem-solving skills and master common hashing patterns used in coding interviews.

🚀 Topics Covered
Frequency Counting using Hash Maps
HashSet for Fast Lookup
Two Sum Pattern (Complement Search)
Prefix Sum + Hashing
String Hashing (Anagrams, Substrings)
Grouping using Hashing
Counting Pairs using Maps

🧠 Key Learnings
Efficient use of unordered_map and unordered_set
Identifying patterns like:
"Have I seen this before?"
"Find complement (target - value)"
"Prefix sum relationships"
Optimizing brute force solutions to O(n)
